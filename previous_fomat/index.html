Projects

Foot Sensor Design and Terrain Classification for Dynamic Locomotion

♦ We designed and fabricated a soft pad with multiple types of sensors embedded in for terrain classification of Cassie Robot, including tactile sensor, acoustic sensor, capacitive sensor, and accelerometers.
♦ A large amount of signal processing and feature extraction was performed, based on which we developed a classification algorithm that had a high classiﬁcation accuracy of 96.7%.
♦ We also completed a number of new experiments by adding multiple noise signals as well as some manual pounding data to prove the redundancy of integrated sensors and the robustness of our trained model.

Robot Grasping in Cluttered Environment with Active Exploration

♦ We design a novel robotic grasping system to automatically pick up objects in a cluttered scene.
♦ Resorting to the active exploration, We utilize a deep Q-Network model to help increase lifting success rate for the suction cup.
See our paper on IROS 2019

Novel Underactuated Robotic Hand for Convenient Grasping

♦ We Proposed a novel underactuated mechanism which can accomplish the linear parallel pinching mode and the self-adaptive grasping mode using only one actuator, switching between the two modes automatically according to the position of the object to be grasped.
